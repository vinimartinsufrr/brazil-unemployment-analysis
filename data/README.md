# Dataset Description

The data used in this project comes from the Brazilian Institute of Geography and Statistics (IBGE),
specifically from the Continuous National Household Sample Survey (PNAD Contínua).

## Indicator
- Unemployment rate (taxa de desocupação)

## Coverage
- Country: Brazil
- Frequency: Quarterly
- Unit: Percentage (%)

## Source
IBGE – PNAD Contínua
